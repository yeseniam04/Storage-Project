# Inventory Management System

## Description
This is a C++ program that allows users to manage inventory items.
It supports adding, removing, saving, and loading items from a file.

## Features
- Add new inventory items
- Delete items
- Save inventory to file
- Load inventory from file

## Technologies Used
- C++
- CLion IDE
- File I/O

## How to Run
1. Clone the repository

2. Open in CLion

3. Run main.cpp

## Usage
The program will prompt the user to:
- Add items (name, quantity, etc.)
- View inventory
- Save/load data
